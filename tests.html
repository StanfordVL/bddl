# Tests ### Overview We provide tests in [test](https://github.com/StanfordVL/iGibson/tree/master/igibson/test). You can run them by this: `bash cd igibson/test pytest --ignore disabled --ignore benchmark ` It will take a few minutes. If all tests pass, you will see something like this ```bash =============================== test session starts ================================ platform linux – Python 3.5.6, pytest-4.6.9, py-1.5.3, pluggy-0.13.1 plugins: openfiles-0.3.0, doctestplus-0.1.3, arraydiff-0.2 collected 27 items
